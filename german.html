'Social distancing' Simulator

Mit diesem Simulator kannst du den Effekt von 'social distancing' auf die Verbreitung einer ansteckenden Krankheit interaktiv austesten. Jede Kugel representiert eine Person, welche entweder gesund (Weiss), immun (Gelb), krank (Rot) oder tot (Grau) ist. Eine gesunde Person wird krank sobald sie mit einer kranken Person in Kontakt kommt. Wenn eine kranke Person die Krankheit übersteht wird sie immun.

Bewege die Regelbalken um die Parameter zu bewegen und beobachte was passiert. Under der Grafik findest du eine detailierte Beschreibung.

'Social distance' (Distanz) 10%
Sterblichkeit 3%
Krankheitsdauer 2.0s
○ Gesunde ● Immune ● Kranke ● Tote Zeit
% % % % s

Social distancing bestimmt in welchem Ausmaß sich die Personein voneinander "distanziert verhalten". Bei einem Wert von 0% gibt es kein distanzierung und die Personen bewegen sich mit Maximalgeschwindigkeit. Somit gibt es unter ihnen viel Kontakt. Bei einem Wert von 100% stehen alle Personen still und es gibt keinerlei Kontakt.

Sterblichkeit bestimmt die Wahrscheinlichkeit, dass eine kranke Person an der Krankheit stirbt. Bei einem Wert von 0% stirbt niemand and der Krankheit. Bei einem Wert von 100% stirbt jeder daran.

Krankheitsdauer bestimmt wie lange eine Person krank bleibt. Umso länger die Dauer, desto mehr Möglichkeiten ergeben sich, dass eine Kranke person weitere Leute ansteckt. Die Animation entspricht einer sehr schnellen Bewegung, aber bloß weil wir die verstrichene Zeit gerne in abgelaufenden Sekunden ausdrücken möchten.

Die Simulation macht einige vereinfachende Annahmen. Darunter die Folgenden: Es gibt keine Inkubationszeit; Personen sind während ihrer gesammten Krankheitsdauer ansteckend; Kontakt führ immer zur einer Ansteckung.

Der Author Andrej Bauer war zu dieser Simulation von einem Artikel der Washington Post von Harry Stevens inspiritiert. Der source code für diese Seite ist frei im GitHub-Projekt social-distancing-simulator verfügbar. Bitte hilf dabei diesen code zu verbessern und übersetze ihn in deine Sprache.
